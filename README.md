# ForecastingStoreSales: Сравнительный анализ AutoML FEDOT и экспертных моделей для прогнозирования временных рядов продаж

▌Описание проекта

Данный репозиторий содержит код и результаты курсовой работы, посвященной сравнительному анализу эффективности фреймворка AutoML FEDOT и ряда экспертных моделей (SARIMAX, LightGBM, MLP) в задаче прогнозирования временных рядов продаж на основе данных Kaggle "Store Sales Time Series Forecasting".

Основная цель проекта — оценить компромисс между достигаемой точностью и трудозатратами при использовании автоматизированных и ручных методов моделирования.

▌Структура проекта

Проект организован для обеспечения полной воспроизводимости экспериментов:

## Воспроизведение результатов

Для воспроизведения всех экспериментов, описанных в работе, выполните следующие шаги:

### 1. Установка окружения

Установите необходимые библиотеки, используя файл requirements.txt:

  pip install -r requirements.txt

### 2. Запуск всех экспериментов (Единая точка входа)

Для последовательного запуска всего цикла исследований — от предобработки данных до финального сравнения моделей — используйте главный скрипт run_all.py.

Внимание: Запуск FEDOT может занять до 5 минут.

  python src/run_all.py

Скрипт run_all.py автоматически выполнит:
1. Обработку и фильтрацию признаков (process_data.py, filter_data.py).
2. Запуск всех экспертных моделей (SARIMA, LightGBM/Optuna, MLP).
3. Запуск экспериментов FEDOT.
4. Сохранение всех метрик и графиков в папку models/.

### 3. Анализ результатов

Результаты всех экспериментов (метрики RMSE, MAE, R²) и сохраненные прогнозы будут доступны в соответствующих подпапках в директории models/.

▌Ключевые выводы
  - SARIMAX (экспертный подход) достиг наивысшей точности (R^2 ≈ 0.872).
  - AutoML FEDOT достиг статистически неотличимого результата (R^2 ≈ 0.867) при минимальных трудозатратах.
  - Исследование timeout показало, что FEDOT находит высококонкурентное решение уже в течение первой минуты.

Вывод: FEDOT является наиболее эффективным и целесообразным решением для промышленного прогнозирования по совокупности факторов "точность/трудозатраты".
